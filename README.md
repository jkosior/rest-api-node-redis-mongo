# REST API

## BUILD AND START API

```
$ make compose_build
```

## START API

```
$ make compose
```

## STOP API

```
$ make down
```

## ENTER CONTAINER

enter server
```
$ make enter_server
```

enter mongo bash / enter mongo shell
```
$ make enter_mongo
$ make enter_mongodb
```

enter redis bash
```
$ make enter_redis
```

## TEST APP

```
$ make tests
```
